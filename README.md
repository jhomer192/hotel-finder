# Hotel Finder

Find hotels near any city using real OpenStreetMap data — no API key, no sign-up, no ads.

**[Try it live →](https://jhomer192.github.io/hotel-finder/)**

## Features

- **Real hotel data** — pulls from OpenStreetMap via Overpass API; not curated lists or sponsored results
- **Star rating filter** — filter by 1–5 stars
- **Sort** — by distance from city center, star rating, or name
- **Map view** — see all results pinned on an interactive map
- **Book links** — each hotel has direct search links to Booking.com and Hotels.com
- **24h cache** — results are cached in sessionStorage so repeat searches are instant

## Tech

React + TypeScript + Vite + Leaflet. Deployed to GitHub Pages.
